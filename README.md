# Task Manager API

A FastAPI-based Task Management API with user authentication and task management features.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Create Tasks
- Update Tasks
- Delete Tasks
- View User Profile
- SQLAlchemy Database Integration
- Pydantic Validation

---

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- JWT Authentication
- Uvicorn
